OOP, Async & Data Handling for AI Systems

NLP Foundations & Text Processing

TensorFlow · HuggingFace

Generative AI, Prompting & Multi-Modal AI
Prompt engineering, LLM parameters (temperature, top-k, top-p), hallucination control, multi-modal foundations (text+vision+audio), GPT vision models, voice AI, speech-to-text.
Azure AI · Copilot

AI Platforms & LangChain Foundations

Advanced RAG

LangChain · Azure OpenAI
11
Agentic AI Foundations
Agent lifecycle, tool use, external API connections, router vs autonomous architectures, single vs multi-agent systems, communication protocols, state management, failure handling.
LangChain · CrewAI · AutoGen
12
Decision Making, Planning & MCP
Goal setting, hierarchical vs reactive planning, MCP fundamentals vs OpenAI function calling, MCP server+client setup, IDE agents, multi-tool orchestration.
Anthropic Claude · AutoGen

Custom Copilots & Multi-Modal Case Study

Azure AI Studio · CrewAI

Memory & Knowledge Retrieval in Agents
Memory architectures (short/long-term, vector vs symbolic, episodic vs semantic), memory compression and summarisation, implementing Agentic RAG, customer support case study.
LangChain · ChromaDB

LangGraph Orchestration & Multi-Agent Systems
LangGraph fundamentals -- state machines for agents, building agent graphs, CrewAI multi-agent orchestration, comparing LangGraph vs CrewAI vs Flowise, resilient workflows.
LangGraph · CrewAI · Flowise

FAQ

Questions
before you apply.

Everything you need to know before applying for your free seat.

I am not a developer. Can I still join?+
Sessions 1--3 are dedicated Python foundations specifically for AI work -- environment setup, data handling, API calls, OOP, and async. If you have basic programming knowledge in any language, you will be ready for Session 4 onwards. You do not need to be a professional developer, but you do need to be comfortable with the idea of writing and running code.
Why is it only RM 1,899 for the founding cohort?+
In the international market, equivalent live instructor-led AI engineering programmes cost between RM 12,000 and RM 75,000. We are charging RM 1,899 -- a nominal fee -- for our founding cohort. Why? Because we are investing in building Malaysia's first generation of AI engineers. You pay enough to show you are serious. We deliver everything else. Cohort 2 will be RM 4,000. This price will never exist again.
Mon--Fri 6pm--10pm is intense. Is this really 4 weeks straight?+
Yes -- 20 sessions over 4 weeks, Mon--Fri, 6pm--10pm. This is a rigorous intensive programme, not a casual course. The depth we cover in 20 sessions -- from Python basics through to deploying a production multi-agent system -- cannot be done in a half-committed way. If you cannot commit to daily attendance, this cohort is not the right fit. If you can, the outcome is transformational.
What if I miss a session?+
All sessions are recorded and you receive lifetime access. We strongly recommend attending every session live -- each session builds on the previous one, and real-time troubleshooting with your instructor is where most of the learning happens. If something unavoidable comes up, the recording is there. But this programme rewards consistent presence.
Does the curriculum use LangChain or LangGraph -- or both?+
Both -- intentionally and in the correct sequence. LangChain is the foundation framework taught in Sessions 7--10, covering chains, tools, RAG, memory, and prompt templates. LangGraph is built on top of LangChain and introduced in Session 15 for orchestrating multi-agent workflows as state machines. You learn LangChain first, then LangGraph -- exactly the right order. The live demo on this page is a LangGraph system running on LangChain tools.
What is the capstone project?+
Sessions 19--20 are your capstone: a fully built and deployed AI Financial Advisor system. It uses 5 specialised agents -- Data Agent (ingests bank statements), Analysis Agent (categorises expenses), Risk Agent (evaluates risk profile), Advisory Agent (suggests investments), and QA Agent (answers natural language queries) -- orchestrated with LangGraph, deployed via FastAPI, with a Streamlit UI. You present it live on Day 20 and receive your certificate.
